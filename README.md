## 📦 como baixar o projeto:

``` bash 

    #Clonar o repositório
    $git clone https://github.com/chagas42/passGenerator

    #Entrar no diretório
    $ cd passGenerator

    #Instalar as dependências
    $ yarn add 

    #Iniciar o projeto
    $ yarn android
```
